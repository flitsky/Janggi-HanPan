    최근 10경기의 자세한 승패 기록(상대, 날짜, 결과)을 3단 테이블 형태의 깔끔한 UI로 확인할 수 있습니다.

🎨 UI & UX 개선 (Improvements)

  • UX 승리 트로피 & 시각적 직관성 강화
    승리 시 🏆 트로피 표기, 진영별(초/한) 색상 구분(파랑/빨강), 결과별 색상 구분(보라/회색)으로 정보를 한눈에 파악할 수 있습니다.
  • UX 스마트 초기 화면
    로그인 여부에 따라 '내 정보(연동 유도)' 또는 '대국(게임 시작)' 탭으로 자동 안내하여 편의성을 높였습니다.

⚙️ 시스템 안정성 (Stability)

몰입감 넘치는 대국 경험

2025년 12월 21일

이번 업데이트에서는 기물을 잡을 때의 시각적 즐거움을 강화하고, 게임의 안정성을 높이는 데 주력했습니다. 더욱 생동감 넘치는 장기 한판을 경험해 보세요!

✨ 주요 변경 사항 (Highlights)

  • New 기물 타격 이펙트 및 애니메이션 추가
    강력한 '쾅' 소리와 함께 잡은 기물이 상대방 진영으로 날아가는 애니메이션이 추가되었습니다. 이제 상대의 기물을 잡는 손맛이 더욱 짜릿해집니다.
  • Improvement 오디오 시스템 대폭 개선
    웹 환경에서 간헐적으로 소리가 끊기던 현상을 해결하고, 효과음(착수음, 타격음 등) 재생 시스템을 최적화했습니다. 이제 어떤 상황에서도 끊김 없는 사운드를 즐길 수 있습니다.

🐛 버그 수정 (Critial Fixes)

  • Fix 왕(King) 제자리 착수 오류 수정
    자신의 왕을 집었다가 제자리에 놓을 경우, 의도치 않게 패배 처리되던 치명적인 오류를 수정했습니다. 이제 안심하고 왕을 조작하세요. (Self-Move Ignore 적용)
  • Fix 게임 루프 오류 해결
    게임이 종료되었음에도 AI가 계속해서 수를 두던 현상(Flip-Flop Bug)을 수정하여 게임 종료 처리를 명확하게 개선했습니다.
  • Fix 기권 시 사운드 누락 수정
    기권(Resign) 버튼을 눌러 패배할 때 패배 효과음이 들리지 않던 문제를 해결했습니다.

⚙️ 기타 변경 사항 (Others)

  • Refactor AI 두뇌 리팩토링: AI 로직을 `AiBrain` 모듈로 통합하여 더욱 일관성 있고 안정적인 수 읽기가 가능해졌습니다.
  • UX 설정 메뉴 개선: '착수음' 옵션 명칭을 '효과음'으로 변경하여 타격음 등 전체 효과음을 제어함을 명확히 했습니다.
  • UX AI 실수 카운트: 대국 종료 후 AI의 실수 횟수를 분석하여 보여주는 기능을 준비했습니다 (내부 로직 적용 완료).
